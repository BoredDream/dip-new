# 我的方法 vs. FF.png(课件经典 DCT 水印)——一致性判断与优劣势分析

> 结论先行:**不一致**。FF.png 画的是课程里教的**经典"全图 DCT + 加性 + 非盲"水印基线**;
> 我的方法是在它的思想之上做了**实质升级**(DWT-DCT 级联 + 盲 QIM + 频带诊断 + 纹理自适应 + Watson JND,
> 并瞄准抗 AI img2img 再生成)。可以把 FF.png 当作**我要超越的基线**,这对答辩反而是个好叙事。

---

## 1. FF.png 里的方法是什么

标题:**"Principles of digital watermark generation based on the DCT algorithm"**。流程:

**嵌入(watermark embedding process):**
```
原始图像 ──DCT变换──┐
                     ├─► DCT 系数组合 ──► 反DCT变换 ──► 含水印的图像
水印图像 ──DCT变换──┘
```
**检测(detection process):**
```
待测图像 ──┐
           ├─► 水印提取 ──► 提取的水印 ──► 水印检测
原始图像 ──┘                 ↑ 检测端需要"原始图像" = 非盲
```

可读出四个关键特征:
1. **只用 DCT**(没有小波/DWT),通常是**整图一次性 DCT**;
2. 水印是**一张图像**(也做 DCT);
3. 嵌入靠 DCT 系数的**加性组合**(典型 `C' = C + α·W`,Cox 扩频式);
4. 检测**需要原始图像**做对比 ⇒ **非盲(non-blind)** 水印。

---

## 2. 和我的方法逐项对照

| 维度 | FF.png(课件基线) | 我的方法(DWT-DCT + 改进) |
|---|---|---|
| 变换 | 仅 DCT,整图一次 | **先 DWT(Haar)取 LL,再对 LL 做 8×8 分块 DCT**(DWT-DCT 级联) |
| 嵌入位置 | 未明确,常为固定中频 | **由 P1 诊断实验选出的"抗再生成幸存低频带"** |
| 水印形态 | 一张**水印图像** | 一串**比特/版权 ID 消息**(可接纠错码 ECC) |
| 嵌入规则 | **加性** `C'=C+αW` | **QIM 量化**(把选定系数量化到代表 0/1 的格点) |
| 提取方式 | **非盲**(需原始图像) | **盲提取**(只需含水印图 + 密钥/步长,无需原图) |
| 不可见性控制 | 固定 α(隐式、靠调参) | **显式 Watson DCT JND + 纹理自适应**(HVS 模型,有理论上限) |
| 目标威胁 | 经典失真(JPEG/噪声) | **低-strength img2img / AI 再生成**(外加经典失真) |

**判断:不是同一个方法**。两者只共享最上层的"用 DCT 频域嵌水印"这一思想,其余环节(变换结构、水印形态、嵌入规则、盲/非盲、可见性模型、目标攻击)都不同。

---

## 3. 我的方法相对 FF.png 的**优势**

1. **盲提取,部署实用**:验证版权时**不需要原始图像**。FF.png 非盲——检测端必须握有原图,现实中往往拿不到,适用面窄。
2. **DWT-DCT 多分辨率**:水印落在小波低频近似(LL)的频域,**抗 JPEG 压缩、抗缩放更稳**;整图单次 DCT 对局部失真/缩放更脆。
3. **盲 QIM 解码简单**:无需原图即可直接判 0/1;加性扩频法通常要"相关检测 + 阈值",且经典实现多为非盲。
4. **不可见性有理论保证(Watson JND)**:可见性不是靠试 α,而是由 HVS 感知模型给每个频带一个**恰可察觉上限**;亮区/纹理区多嵌、平坦区少嵌,主观质量更优、更不易出块效应。
5. **嵌入位置数据驱动**:用诊断实验(频带×img2img 强度的破坏热图)选嵌入带,**针对性抵抗 AI 再生成**;FF.png 的固定带没有任何针对 AI 编辑的设计。
6. **纹理自适应强度**:在鲁棒性与不可见性之间取得更好的折中。
7. **承载消息比特 + 可纠错**:能完整恢复版权 ID,且 BER/比特准确率**可量化、可画曲线**;嵌图像式水印难以量化、难纠错。

---

## 4. 我的方法相对 FF.png 的**劣势**

1. **容量更低**:QIM 每块/每带只嵌 1 bit,总容量远小于"嵌入一整张水印图像"。若需求是藏一张可视 logo,FF.png 反而合适。
2. **对全局幅度/对比度变化更敏感**:QIM 量化步长固定,遇到直方图拉伸、gamma、整体亮度缩放时格点会错位、掉比特;加性 + 非盲法借助原图归一化,对幅度缩放更稳。
3. **自适应步长有一致性风险**:Δ 依赖块统计,嵌入端与提取端必须**重算一致**,实现更复杂;固定 α 的加性法没有这个问题(更"皮实")。
4. **展示不如可视水印直观**:FF.png 能"提取出一张看得见的水印图",答辩演示冲击力强;我的是不可见比特,得靠 BitAcc/曲线图说话。
5. **几何攻击(旋转/平移/裁剪)仍脆**:分块对齐被破坏即失效——这是分块变换域水印的通病(FF.png 的整图 DCT 同样脆,此项算两者共同短板,非我独有劣势)。
6. **概念与讲解更重**:三处改进 + JND 模型,环节多;FF.png 一张图就能讲完。

---

## 5. 给答辩的建议

把 FF.png 明确定位为**"课程教学基线"**,然后用一句话串起整个项目动机:

> "课件里的经典 DCT 水印是**非盲、固定频带、无感知模型、未考虑 AI 再生成**的。
> 我们以它为基线,逐一改进:**升级为 DWT-DCT 盲量化**(去掉对原图的依赖)、
> **用诊断实验选抗再生成的频带**、**用 Watson JND + 纹理自适应保证不可见**,
> 最终在低-strength img2img 攻击下显著优于该基线。"

这样 FF.png 不是"和你冲突的方法",而是**你论证改进价值的对照起点**——劣势项(容量、抗幅度缩放)则在报告"局限与展望"里诚实列出,反而显得严谨。
