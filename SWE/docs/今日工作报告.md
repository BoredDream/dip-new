## 2. 核心方法关系(给不熟项目的人)

- **经典 / 改进版水印**(主角):手工设计的数学变换(DWT+DCT+QIM 量化嵌入),**不需要训练**。
  我们今天的主创新 `ImprovedDwtDct` 属于这类。
- **深度潜空间水印**(配角=上界基线):神经网络学出来的水印(VINE/RoSteALS 路线),**需要训练**。
  它只是"天花板参考",回答"既然要抗 AI,为何不直接上深度"。

`ImprovedDwtDct` 的三处可解释改进(对应方案 §4.4,可独立开关做消融):
1. **多低频带**:嵌入位置从默认中频 #18 → P1 诊断选出的低频幸存带 #1–7,多频带+多块重复+多数表决;
2. **纹理自适应**:步长按块纹理活动度缩放(纹理强多嵌、平坦少嵌);
3. **Watson JND**:用感知模型给每频带"恰不可察觉"上限,压低可见性。
> 关键工程点:自适应步长只用**嵌入后稳定**的统计量(块均值=DC、非载体高频能量)推算,不自指被嵌系数 →
> 盲提取干净准确率 = 1.0(单测卡死)。

## 3. 关键结果(20 张真实图实测)

### 3.1 消融:改进逐档有效(全攻击均值比特准确率)
```
默认 DwtDct  →  +多频带  →  +纹理  →  +JND
   0.638        0.764      0.833     0.784
```
- **多频带 + 纹理是主要增益来源**(0.64→0.83);
- **JND 档略降鲁棒性(0.833→0.784)是预期的**——JND 换的是**可见性**不是鲁棒性(它把嵌入预算往不敏感频带挪)。
- 图:`results/figures/ablation.png`、各 `sweep_<attack>.png`。

### 3.2 真扩散 img2img(本地 SD1.5/GPU,真 held-out)—— 最有分量的结论
真扩散下各方法留存率(strength=0.1 / 0.2 / 0.3):

| 方法 | 0.1 | 0.2 | 0.3 | 结论 |
|---|---|---|---|---|
| **Deep-Latent(深度上界)** | **0.96** | **0.95** | **0.92** | **稳稳幸存** |
| SpreadSpec(下界,但 31dB 可见) | 0.68 | 0.59 | 0.57 | 部分残留 |
| DwtDct-default / Imp+multiband / Imp+texture / Imp+JND | ≈0.50 | ≈0.50 | ≈0.50 | **全部阵亡** |
| DCT-QIM-JPEG | 0.53 | 0.56 | 0.52 | 阵亡 |

> 深度模型为**正经训练**(冻结 SD-VAE + CNN 解码器,GPU,训练集 DIV2K 100 张**与测试 22 张完全不重叠**):
> 在 20 张 held-out 图上 clean_acc=**0.978**(泛化好,不再像早期 2 图过拟合版的 0.85),真扩散下稳在 0.92–0.96。
> **代价**:深度水印 PSNR≈23dB / SSIM≈0.72,**画质明显不如经典**(经典 49–66dB)——鲁棒性换可见性,这是诚实 tradeoff。

**三条结论(诚实,可写进报告):**
1. **真扩散下,所有 QIM 类水印(含改进版)在 strength=0.1 就集体跌到 ~0.5(死)** —— 印证 Zhao et al.
   (NeurIPS 2024)"像素/变换域隐形水印可被再生成证明性移除"。
2. **改进版相对默认的优势在真扩散下消失**:真 img2img 连低频内容都重写,把低频带嵌入也洗掉。
   → 改进版的增益是针对**压缩/模糊/低通退化**的,不是针对真生成式重绘的。
3. **只有深度水印幸存**,且这次是真 held-out(深度模型从没见过真扩散),清晰兑现"深度抗再生成"。
- 图:`results/figures/sweep_diffusion_img2img.png`、`robustness_table_ai.png`。

### 3.3 改进的意义(答辩话术,详见 `docs/EVALUATION.md §6`)
- 改进版在**最常见威胁(压缩/编辑)下真的更鲁棒且更隐形**(PSNR≈49dB);
- 把经典方法做到极致后**仍被真扩散通杀**,反而**严格论证了深度水印的必要性**(一个被逼到极限仍失败的对照,
  比随手弱对照值钱);
- 三条曲线一起画出攻防分界:**经典抗压缩 → 改进后逼近 → 深度抗再生成**。

## 5. 诚实的局限(必须写进报告,别夸大)

- **深度上界已正经训练但仍偏 PoC**:已用冻结 SD-VAE + CNN 解码器 + GPU + DIV2K(与测试不重叠)训成,
  泛化与抗真扩散都可信(clean 0.978、真扩散 0.92–0.96)。但**画质偏低(PSNR≈23dB)**,且训练集仅 100 张、128px;
  论文级还可换 ConvNeXt 解码器 + 256px + 更多数据 + 更长训练把画质提上去。
- **样本量**:20 张做趋势够,正式定稿建议 ~100 张。
- **`Imp+JND` 的 PSNR 出现 inf**:个别近乎纯色的图(如 page/text)上嵌入退化(clean_acc 0.97<1.0),是边界情形,
  需在报告里说明或对纯色块加保护。
- 几何攻击(裁剪/旋转)QIM 类注定弱,属构造性失败,不直接拿来比较。
