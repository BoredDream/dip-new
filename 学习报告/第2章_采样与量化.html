CHAPTER 2 · SAMPLING & QUANTIZATION

第 2 章　采样与量化

回答一个核心问题：连续的真实世界，如何变成离散的数字矩阵？再讲图像的几何变换。
📌 本章学习目标
  • 理解成像模型（人眼 ≈ 相机）与四个坐标系
  • 分清采样（管分辨率）与量化（管灰度级）
  • 掌握存储量公式与「采样数/灰度级 ↔ 图像质量」的权衡（重点）
  • 用 3×3 齐次矩阵统一表达平移/缩放/旋转/仿射，理解插值

2.1 成像模型

人眼 ≈ 相机

人眼相机作用
角膜+晶状体镜头 lens聚焦光线
瞳孔/虹膜光圈 aperture控制进光量
视网膜 retina传感器 sensor捕获图像
视神经数据线传输图像

视网膜上有两类感光细胞：视锥细胞（集中在中央凹，负责彩色与细节，对应明视觉）和视杆细胞（外周多，负责暗视觉与亮度）。这解释了人眼对亮度/细节的感知规律，也是后续「为什么图像增强对人有效」的生理基础。

视觉错觉 = 人眼不是「老实」的传感器
马赫带（边界处亮度被夸大）、同时对比（同一灰块在不同背景下显得不同）、各种几何错觉——说明 人感知的亮度 ≠ 物理亮度。图像处理的目标常常是「让人看着更好」，而非「物理上更准」。

成像传感器的三种方式

  • 单个传感器：配合机械移动逐点扫描成像；
  • 线传感器（传感器带/环）：一次出一行，配合线性移动；典型是 CT（X 射线绕物体旋转 + 平移 → 图像重建出断层）；
  • 面阵传感器：一次成整幅图，即相机的 CCD/CMOS。

透视投影与坐标系

3D 世界点投影到 2D 图像，要经过四个坐标系的串联变换：

世界坐标系 → 相机坐标系 → 图像平面坐标系 → 数字图像坐标系

图像捕获两类信息：几何信息（位置、点、线、曲线）与 光度信息（强度、颜色）。相机模型是对真实 3D→2D 关系的近似。

2.2 采样与量化（本章核心）

两个独立的离散化过程
  • 采样 Sampling：对空间坐标 (x,y) 离散化 → 决定分辨率（图像有 $M\times N$ 个像素）。采样越密，像素越多。
  • 量化 Quantization：对幅度（灰度值） 离散化 → 决定灰度级。每像素用 $k$ 位表示，共 $L=2^k$ 个灰度级（8 位 = 256 级）。
一句话：采样管「横竖分多少格」，量化管「每格的颜色分多少档」。

存储量公式

$$b = M\times N\times k \quad\text{（方形图 }M=N\text{ 时 } b=N^2 k\text{）}$$

其中 $k$ = 每像素比特数，$L=2^k$ = 灰度级数。这条公式说明分辨率和位深都直接决定存储/带宽——这也是后面「图像编码压缩」的动机。

⚠️ 质量权衡（本章最常考）
固定项减少另一项退化现象
固定灰度级 $k$减少采样点 $N$方块效应 blocking（图像变马赛克）
固定采样 $N$减少灰度级 $k$假轮廓 false contour（平坦区出现条带，极端为二值图）
实用结论：当图像尺寸/容量受限时——
  • 图像直方图平坦、细节少（如人脸）→ 粗采样 + 细量化，避免假轮廓；
  • 图像细节丰富（如看台人群）→ 细采样 + 粗量化，避免模糊。
口诀：缺细节就多给采样，缺层次就多给灰度。

2.3 图像几何变换

用 3×3 齐次坐标矩阵 统一表达所有线性几何变换。点写成 $v=[x,\ y,\ 1]^{T}$，变换为 $v' = Av$。

平移 Translation

$$T=\begin{bmatrix}1&0&t_x\\0&1&t_y\\0&0&1\end{bmatrix},\qquad T^{-1}=\begin{bmatrix}1&0&-t_x\\0&1&-t_y\\0&0&1\end{bmatrix}$$

缩放 Scaling

$$S=\begin{bmatrix}s_x&0&0\\0&s_y&0\\0&0&1\end{bmatrix},\qquad S^{-1}=\begin{bmatrix}1/s_x&0&0\\0&1/s_y&0\\0&0&1\end{bmatrix}$$

旋转 Rotation（绕原点，角度 $\theta$）

$$R=\begin{bmatrix}\cos\theta&\sin\theta&0\\-\sin\theta&\cos\theta&0\\0&0&1\end{bmatrix},\qquad R^{-1}=\begin{bmatrix}\cos\theta&-\sin\theta&0\\\sin\theta&\cos\theta&0\\0&0&1\end{bmatrix}$$

仿射 Affine

$$A=\begin{bmatrix}a_{11}&a_{12}&t_x\\a_{21}&a_{22}&t_y\\0&0&1\end{bmatrix}$$

仿射 = 平移 + 缩放 + 旋转 + 错切的组合，保持「直线仍是直线、平行线仍平行」。

插值（变换后的必经步骤）
缩放/旋转后，目标像素映射回原图常落在非整数位置，需要插值估计灰度：
  • 最近邻插值：取最近像素，快但有锯齿/块状；
  • 双线性插值：用周围 4 个像素加权平均，平滑但略糊。
🎯 与水印项目的关联
本章是几何攻击的理论来源。缩放、旋转、裁剪是水印鲁棒性测试的标准项；而插值带来的取整误差正是 LSB（最低有效位）类方法极易被破坏的原因——只要图像被重采样一次，最低位几乎全乱。这解释了为什么你的 traditional_watermark.py 里变换域方法（DCT/小波/SVD）比 LSB 抗几何攻击更强。
🧠 一句话小结
采样定分辨率、量化定灰度级，存储量 $b=MNk$；细节少→细量化、细节多→细采样；几何变换统一为 $v'=Av$，变换后需插值。
