概念驱动方式代表

6.2 神经网络

神经元（基本单元）

一个神经元做的事：对输入加权求和、加偏置、过激活函数：

$$a=\sigma\!\left(\mathbf{w}^{T}\mathbf{x}+b\right)$$

其中 $\mathbf{w}$ 是权重、$b$ 是偏置、$\sigma$ 是激活函数（如 Sigmoid、ReLU）。激活函数提供非线性，否则多层网络等价于一层。

全连接网络 & 「深度」

把神经元分层连接：输入层 → 多个隐藏层 → 输出层，每层每个神经元与下层全部相连。层数多了就叫「深」——这就是 Deep Learning 名字的由来。

6.3 卷积神经网络 CNN

CNN 为什么适合图像：两大特性
  • 局部连接：每个神经元只看输入图的一个局部窗口（感受野），而非整张图——符合「图像的局部相关性」；
  • 权值共享：同一个卷积核在整张图上滑动复用同一组权重——参数量大幅减少，且具备平移不变性。
对比第 3 章：CNN 的卷积核 = 可学习的滤波模板。Sobel/拉普拉斯是手工设计的核，CNN 的核是训练学出来的。

卷积操作

对彩色图（如 $32\times32\times3$），用一个卷积核（如 $5\times5\times3$）在空间上滑动，每个位置算 $\mathbf{w}^{T}\mathbf{x}+b$ 得一个实数，所有位置拼成输出特征图。

典型 CNN 结构

CONV（卷积层）→ RELU（激活层）→ POOL（池化层）→ … → FC（全连接层）→ 分类输出

  • CONV：提取局部特征（浅层学边缘/纹理，深层学语义）；
  • RELU：非线性激活 $\max(0,x)$；
  • POOL：下采样（如最大池化），降维 + 增强平移鲁棒性；
  • FC：把特征汇总，输出各类别得分。

6.4 神经网络训练（闭环）

输入图像 X │ ▼ 图像表示 ──► 分类模型(CNN) ──► 预测值 Ŷ ┐ ▲ ├─► 损失函数 ◄── 真实值 Y │ ┘ │ 模型参数 ◄── 优化算法 ◄────────────────────────┘ 损失值
环节关键知识点
分类模型全连接网络、CNN、激活函数
损失函数Softmax + 交叉熵、SVM 合页损失、其他损失
优化梯度下降 + 反向传播、动量、自适应梯度（Adam 等）
训练技巧数据增强、权值初始化、批归一化 BN、Dropout、超参调优

核心闭环：前向算出预测 $\hat Y$ → 与真值 $Y$ 算损失 → 反向传播求梯度 → 优化器更新参数 → 循环，直到损失收敛。

6.5 四大计算机视觉任务（重点）

任务输出特点
分类 Classification整图一个类别（「猫」）无空间位置
语义分割 Semantic Segmentation每个像素的类别（草/猫/树/天）只分类不分实例
目标检测 Object Detection每个物体的类别 + 边界框多目标、给框
实例分割 Instance Segmentation像素级 + 区分每个实例（狗1/狗2/猫）多目标、像素级
🎯 与水印项目的关联（关键一章）
  • 课题要求「图片经 AI 修图 后水印仍留存」。AI 修图的本质是扩散模型重绘——一种生成式深度网络。理解 CNN/深度学习，才能理解为什么传统频域水印会被 AI 重绘抹掉：重绘相当于把图像「打散重画」，频域系数被大幅改写。
  • VINE 参考实现的核心思路正是本章的产物：用「模糊」作为 AI 编辑的代理攻击，联合训练 编码器 + 攻击层 + 解码器（端到端深度学习），让水印学会扛过一次扩散重绘。这就是把「水印嵌入/提取」也变成可训练的神经网络。
  • 你的对照实验 = 传统手工方法（第 4 章频域）vs 学习式方法（本章）。理解两者，才能在论文里讲清楚「为什么深度方法对 AI 攻击更鲁棒」。
🧠 一句话小结
AI⊃ML⊃DL；神经元 = $\sigma(\mathbf{w}^T\mathbf{x}+b)$，多层即「深」。CNN 靠局部连接 + 权值共享高效处理图像，卷积核 = 可学习的滤波器。训练 = 前向→损失→反向传播→更新的闭环。视觉四大任务：分类 / 语义分割 / 检测 / 实例分割。
